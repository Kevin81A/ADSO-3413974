# 🦸‍♂️ GitHub Comic: Cómo subir y bajar archivos

## 🎬 Escena 1 – El problema

👩‍💻 **Ana:**

> ¡Terminé mi proyecto! Pero está solo en mi computadora 😭

🧑‍🚀 **GitHero:**

> ¡No te preocupes! Vamos a usar **GitHub** para guardarlo en la nube.

---

## 🎬 Escena 2 – Crear el repositorio

🧑‍🚀 **GitHero:**

> Primero crea un repositorio.

1. Ir a GitHub
2. Hacer clic en **New Repository**
3. Escribir el nombre del proyecto

Ejemplo:

```
mi-proyecto
```

🎉 ¡Repositorio creado!

---

## 🎬 Escena 3 – Preparar el proyecto

👩‍💻 **Ana:**

> Tengo mi carpeta del proyecto lista.

🧑‍🚀 **GitHero:**

> Abre la terminal en la carpeta y escribe:

```
git init
```

💬 Esto inicia Git en tu proyecto.

---

## 🎬 Escena 4 – Guardar cambios

🧑‍🚀 **GitHero:**

> Ahora vamos a guardar los archivos.

```
git add .
git commit -m "Primer commit"
```

💡

* `git add .` prepara los archivos
* `git commit` guarda los cambios

---

## 🎬 Escena 5 – Subir a GitHub

👩‍💻 **Ana:**

> ¡Quiero subirlo!

🧑‍🚀 **GitHero:**

```
git remote add origin https://github.com/usuario/mi-proyecto.git
git push -u origin main
```

🚀 ¡Tu proyecto ya está en GitHub!

---

## 🎬 Escena 6 – Descargar un proyecto

👩‍💻 **Ana:**

> ¿Cómo descargo un proyecto?

🧑‍🚀 **GitHero:**

```
git clone https://github.com/usuario/mi-proyecto.git
```

💻 Esto descargará todo el proyecto.

---

## 🎬 Escena 7 – Actualizar cambios

👩‍💻 **Ana:**

> ¿Y si alguien actualizó el proyecto?

🧑‍🚀 **GitHero:**

```
git pull
```

⚡ Esto trae los cambios nuevos.

---

# 🏁 Fin del cómic

## Poderes que aprendiste

| Comando    | Poder               |
| ---------- | ------------------- |
| git add    | preparar archivos   |
| git commit | guardar cambios     |
| git push   | subir archivos      |
| git clone  | descargar proyecto  |
| git pull   | actualizar proyecto |

🦸‍♂️ ¡Ahora eres un héroe de GitHub!
